# 🛠 DEEIX 极速部署与一键更新

请根据您的操作系统选择相应命令在终端中运行。脚本支持**初次安装**、**更新升级（保留配置）**及**服务启停管理**。

## 🐧 Linux / 🍏 macOS / 🤖 Android (Termux)
打开您的终端，执行以下一行命令即可：
```bash
curl -fsSL https://raw.githubusercontent.com/1474443844/Build/main/deeix/deploy.sh | bash
```


## 🪟 Windows (Powershell)
请以 **管理员身份** 打开 PowerShell（右键以管理员身份运行），并输入以下命令：
```powershell
irm https://raw.githubusercontent.com/1474443844/Build/main/deeix/deploy.ps1 | iex
```
